# Prompt: Create a news post

Use this prompt to create a new news post.

---

Please write a news post for the DataverseNO information website based on the following information:

[DESCRIBE THE NEWS HERE]

The post should:
- Be written in clear, accessible English
- Be concise (100–300 words)
- Use the following frontmatter template:

```yaml
---
date: YYYY-MM-DD
authors:
  - name: DataverseNO
tags:
  - [relevant tags]
---
```

Save the file as `docs/en/news/posts/YYYY-MM-DD-short-title.md` and create a corresponding Norwegian version at `docs/nn/news/posts/YYYY-MM-DD-short-title.md`.
