# Metadata

Review the dataset's metadata for completeness, accuracy, and discoverability.

## Required fields

Verify all required fields are filled in:

| Field | What to check |
|---|---|
| Title | Descriptive, not generic ("Dataset 1" is not acceptable) |
| Author | All contributors listed with affiliations |
| Contact | Valid email address |
| Description | Sufficient detail to understand the data without opening files |
| Subject | Correct category selected |
| Keyword | At least 3–5 relevant keywords |

## Recommended fields

Encourage depositors to fill in:

- Related publication (link to the article using this data)
- Geographic coverage
- Time period covered
- Language of the data

## Common issues

- Title too vague or identical to the article title
- Description copied from the abstract — should describe the *data*, not the study
- Missing or incorrect author affiliations
- No keywords, or keywords too generic
