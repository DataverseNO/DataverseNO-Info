# Publish dataset

When the dataset meets all requirements, publish it.

## Pre-publication checklist

- [ ] All required metadata fields complete and accurate
- [ ] README present and informative
- [ ] Files in open formats (or open equivalents provided)
- [ ] No sensitive data
- [ ] Appropriate license applied
- [ ] Related publication linked (if applicable)

## Publishing

1. Confirm all items in the checklist above
2. Click **Publish Dataset** in DataverseNO
3. Select the version type (Major for first publication)
4. Confirm publication

A DOI is assigned at this point and the dataset becomes publicly accessible.

## After publication

The depositor receives a notification. The dataset is now:

- Publicly accessible at its DOI
- Indexed by search engines
- Citable in publications
