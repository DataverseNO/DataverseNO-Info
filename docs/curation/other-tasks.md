# Other curation tasks

## Anonymisation and double-blind peer review

If a depositor needs their dataset anonymised for double-blind peer review, DataverseNO supports private URLs that allow reviewers to access the dataset without it being publicly visible.

To set up a private URL: navigate to the dataset → **Edit Dataset → Private URL**.

Share this URL with the journal or depositor for reviewer access.

## Long-term preservation

DataverseNO commits to long-term preservation of all published datasets. As curator, your role in preservation is to ensure:

- Files are in formats suitable for long-term access (prefer open formats)
- Metadata is complete enough to understand the data without additional context

Formal preservation activities (format migration, integrity checks) are handled by the repository operations team.

## New curator training

New curators shadow an experienced curator on their first several datasets before taking on independent curation. Contact the repository manager to arrange this.
