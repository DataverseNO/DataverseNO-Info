# New information website for DataverseNO

We are pleased to announce the launch of the redesigned DataverseNO information website. The new site is built on [MkDocs](https://www.mkdocs.org/) with the [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/) theme and is hosted on GitHub Pages.

The new site aims to provide a better experience for depositors and repository staff, with improved navigation, search, and bilingual support (Norwegian and English).

If you have feedback on the new website, please contact [support@dataverse.no](mailto:support@dataverse.no).
