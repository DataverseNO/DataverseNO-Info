# Curation Guidelines

These guidelines describe the curation process for DataverseNO curators — the steps involved in reviewing, improving, and publishing submitted datasets.

## What is curation?

Curation is a quality review of a submitted dataset. The curator checks that the data and metadata meet DataverseNO's requirements and community standards, and communicates with the depositor if improvements are needed.

Curation is not gatekeeping — it is a support service that helps depositors publish better, more reusable data.

## Curation workflow

1. Dataset submitted by depositor
2. Curator assigned and notified
3. Curator reviews [metadata](metadata.md), [files](files.md), and [terms](terms.md)
4. Curator either [returns dataset to the author](return-to-author.md) with feedback, or [publishes](publish.md) the dataset
