# Curation Labels

DataverseNO uses curation labels to track the status of datasets during the curation process. Labels help curators and collection managers understand where a dataset is in the workflow at a glance.

!!! note "Content to be added"
    This page is under development. Detailed descriptions of all curation labels and how to apply them will be added here.

## Overview

Labels are applied to datasets in the DataverseNO interface and are visible to curators and collection managers (not to depositors).

Contact [support@dataverse.no](mailto:support@dataverse.no) for questions about curation labels.
