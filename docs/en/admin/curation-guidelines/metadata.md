# Metadata

Review metadata for completeness, accuracy, and discoverability. All metadata should be in English or standard scientific language.

First check the **Versions** tab to determine whether this is a new dataset or an updated version of a previously published one.

## Field-by-field checklist

| Field | What to verify |
|---|---|
| **Title** | Descriptive and specific — not "Dataset 1" or a copy of the article title |
| **Author** | All contributors listed; names inverted (Family, Given); ORCID encouraged |
| **Contact** | Valid institutional email address |
| **Description** | Explains what the data *contains* (not a copy of the article abstract); formatted with `<p>` tags |
| **Subject** | Correct category selected |
| **Keyword** | Each keyword in a separate field; at least a few relevant terms |
| **Related publication** | Linked to published article, or "Submitted for review" noted if manuscript is under review |
| **Language** | Represents the *language of analysis*, not the subject language |
| **Producer** | Depositor's institution or funding organisation; for students, use institution or funder — not the student's own name |

## Common issues

- Description copied from the article abstract — it should describe the *data*, not the study
- Author names not inverted, or ORCID missing
- "Related publication" left empty when an article exists
- Keywords too generic or all entered in one field instead of separately
- Language field set to the language spoken in recordings rather than the analysis language

## Recommended fields

Encourage depositors to also complete:

- Geographic coverage
- Time period covered
- Funding information
