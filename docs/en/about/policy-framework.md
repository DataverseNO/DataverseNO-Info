# DataverseNO Policy Framework

The DataverseNO Policy Framework describes the principles, policies, and responsibilities that govern the operation and use of the repository. Together, these documents help ensure that DataverseNO provides trustworthy, sustainable, and FAIR-enabling management, preservation, and sharing of research data.

## Key documents

| Document | Description |
|---|---|
| [DataverseNO Organizational Agreement](https://site.uit.no/dataverseno/organizational-agreement/) | The common agreement between partner institutions governing shared governance and repository practices |
| [Deposit guidelines](../deposit/index.md) | Requirements and procedures for depositing data in DataverseNO |
| [Preservation Policy](https://site.uit.no/dataverseno/preservation-policy/) | Describes how DataverseNO ensures long-term preservation of published datasets |
| [Access and Reuse Policy](https://site.uit.no/dataverseno/access-policy/) | Terms for accessing and reusing datasets |

## CoreTrustSeal certification

DataverseNO was [CoreTrustSeal](https://www.coretrustseal.org/) certified in 2020, demonstrating that it meets the requirements for trustworthy and sustainable digital repositories. The repository is currently in the process of renewing its certification.

For more information, contact [support@dataverse.no](mailto:support@dataverse.no).
