# What is DataverseNO?

[DataverseNO](https://dataverse.no/) is a national repository for open research data. It enables researchers to archive, publish, preserve, and share datasets in a way that makes them easier to find, access, understand, and reuse.

DataverseNO hosts research data from all disciplines, but currently only accepts datasets that can be shared **openly**. This means that data requiring protection — such as personal data, health data, or other sensitive information — must be appropriately managed before it can be deposited. If you are unsure whether your data can be shared through DataverseNO, please [contact your local support service](https://site.uit.no/dataverseno/contact/).

A distinctive feature of DataverseNO is that all datasets are curated by trained support staff (curators) before publication. Curators provide dataset-specific guidance on metadata, documentation, file formats, licensing, and other aspects that contribute to making data more **F**indable, **A**ccessible, **I**nteroperable and **R**eusable ([FAIR](https://www.go-fair.org/fair-principles/)).

DataverseNO is operated by **UiT The Arctic University of Norway** on behalf of a national consortium of partner institutions. The repository is organised into institutional collections, allowing participating institutions to manage and showcase their research outputs while following common repository policies and standards. Institutional collections may also contain sub-collections for research centres, projects, or other organisational units.

DataverseNO is operated according to a common agreement (the [DataverseNO Organizational Agreement](https://site.uit.no/dataverseno/organizational-agreement/)) between the partner institutions, ensuring shared governance and consistent repository practices.

The repository is based on the open-source platform [Dataverse](https://dataverse.org/), a widely used research data repository solution developed by an international community led by Harvard University.

## Key facts

| | |
|---|---|
| Platform | Harvard Dataverse |
| Operator | UiT The Arctic University of Norway |
| Access | Open to researchers at Norwegian institutions |
| Cost | Free |
| Data types | All research disciplines (openly shareable data only) |
| Persistent identifier | DOI assigned to all published datasets |
| Certification | CoreTrustSeal certified since 2020 |
