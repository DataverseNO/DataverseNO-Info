# Prepare your data

Before uploading to DataverseNO, prepare your files and documentation so your dataset is understandable and reusable by others.

## File formats

Use open, non-proprietary formats where possible. Preferred formats include:

| Data type | Preferred format |
|---|---|
| Tabular data | CSV, TSV |
| Text | TXT, PDF/A |
| Images | TIFF, PNG |
| Audio | WAV, FLAC |
| Video | MP4 (H.264) |
| Geospatial | GeoTIFF, Shapefile, GeoJSON |

Proprietary formats (Excel, SPSS, etc.) are accepted alongside open equivalents.

## Documentation

Include a `README` file describing:

- What the data contains
- How it was collected
- Any processing or cleaning applied
- How to interpret variable names and values
- Any software required to open the files

## Sensitive data

Remove any personal, sensitive, or confidential information before depositing. DataverseNO does not accept data requiring protection.

!!! warning "Check before you deposit"
    If you are unsure whether your data contains sensitive information, contact your institution's data protection officer before proceeding.
