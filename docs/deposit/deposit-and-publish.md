# Deposit and publish your data

## Log in to DataverseNO

Go to [dataverse.no](https://dataverse.no) and log in with your institutional credentials via Feide.

## Create a new dataset

1. Navigate to your institution's collection (or the root DataverseNO collection if unsure)
2. Click **Add Data → New Dataset**
3. Fill in the required metadata fields

## Required metadata

| Field | Description |
|---|---|
| Title | Descriptive title for the dataset |
| Author | Name(s) and affiliation(s) of all contributors |
| Contact | Email address for dataset inquiries |
| Description | Summary of what the data contains |
| Subject | Research area (select from list) |
| Keyword | Relevant terms to improve discoverability |

## Upload your files

Upload your data files and documentation (README). There is no file size limit for individual files.

## Submit for review

When ready, click **Submit for Review**. Your dataset will be sent to a DataverseNO curator for review before publication.

!!! note "Curation"
    Curation is a quality check — the curator may contact you with suggestions. This is normal and helps ensure your dataset meets community standards.
