# What is DataverseNO?

DataverseNO is a national repository for open research data. It is based on the [Dataverse](https://dataverse.org/) platform, developed by Harvard University.

The repository allows researchers to deposit, publish, and share research data openly, in line with the FAIR principles (Findable, Accessible, Interoperable, Reusable).

DataverseNO is hosted and operated by UiT The Arctic University of Norway, and is available to researchers at all Norwegian higher education and research institutions.

## Key facts

| | |
|---|---|
| Platform | Harvard Dataverse |
| Operator | UiT The Arctic University of Norway |
| Access | Open to Norwegian researchers |
| Cost | Free |
| Data types | All research data types accepted |
| License | Open access (CC licenses) |
| Persistent identifier | DOI assigned to all published datasets |
